# Komently --- AI-Ready Universal Comment System

Komently is a universal comment system API designed for third-party
websites. It provides a complete infrastructure for comment sections
including database, authentication, file hosting, and a modern
Reddit-style UI via an embeddable SDK.

This project is a draft working website built with Next.js that
demonstrates Komently's core concept and prepares the foundation for
future AI-agent integration.

------------------------------------------------------------------------

## 🚀 Project Purpose

The goal of Komently is to allow website owners (users) to easily add a
powerful, scalable comment system to their platforms without building
backend infrastructure from scratch.

This repository contains:

-   Marketing/demo website
-   Mock dashboard
-   SDK demo integration
-   AI agent integration plan (future work)

------------------------------------------------------------------------

## 👥 Target Users

### Primary Users

-   Website owners
-   Indie developers
-   Content platforms
-   Blog owners

### Secondary Users (Commenters)

-   Visitors who interact with comment sections
-   Authenticated or guest users

------------------------------------------------------------------------

## ✨ Core Features (Draft)

-   Universal comment section with public ID
-   Reddit-style threaded UI
-   Like / dislike system
-   Pagination and sorting
-   Guest and authenticated commenting (planned)
-   Section management dashboard (mock)
-   AI moderation planning (not yet implemented)

------------------------------------------------------------------------

## 🧠 Planned AI Agent (Future Work)

Komently will integrate an AI Moderator Agent that will:

-   Detect spam and toxic comments
-   Assist moderators
-   Improve smart reply fetching
-   Provide discussion insights

The AI is not implemented yet --- this project focuses on system design
readiness.
