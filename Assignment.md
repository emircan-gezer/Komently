Homework 2: Website Development & AI Agent Planning

Yönergeler
The objective of this assignment is to design and implement a draft working website on a topic of your choice and to plan the future integration of an AI agent into this website.
This homework focuses on web development fundamentals, system planning, and documentation, which will serve as the foundation for later AI-agent–based assignments.



You are required to:

Choose a topic of your interest (e.g., education, health, transportation, e-commerce, smart cities, entertainment, personal portfolio, etc.).

Develop a draft working website related to this topic.

Create a planning document describing how an AI agent will be integrated into your website in future assignments.

Publish your project on GitHub and submit the repository link.

This assignment emphasizes design clarity, functionality, documentation quality, and forward-thinking system design, rather than advanced AI implementation.

Part 1 – Website Development (Draft Version)
Your website must meet the following minimum requirements:

Functional Requirements
A working website with at least:

A home page

At least two additional pages or sections

Basic navigation (menu, links, or routing)

Static and dynamic content 

Technical Requirements
You may use any web technology (e.g., HTML/CSS/JavaScript, React, Vue, Angular, Django, Flask, etc.).

The website must run locally and be deployable (e.g., GitHub Pages, Vercel, Netlify).

The project must be version-controlled using Git.

Part 2 – AI Agent Planning Document
You must prepare a short written document (2–4 pages) that includes the following sections:

1. Project Overview
Website topic and purpose

Target users

Core features of the website

2. AI Agent Concept
Describe the AI agent you plan to integrate in later assignments:

What problem will the AI agent solve?

What type of agent will it be? (e.g., assistant, recommender, advisor, evaluator)

How users will interact with the agent (chat, forms, background automation, etc.)

3. System Architecture (High-Level)
How the AI agent will interact with:

Frontend

Backend (if any)

External APIs or services

A simple architecture diagram is encouraged 

Part 3 – GitHub Repository
Your GitHub repository must include:

All source code of the website

A README.md file containing:

Project title and description

Setup and run instructions

Technologies used

Link to the planning document (or included as a PDF/Markdown file)

Submission Requirements
Students must submit:

GitHub repository link

Planning document (PDF)

Live demo link 

Notes
This is a foundational assignment. The AI agent does not need to be implemented yet.

Your design choices will directly impact future homework assignments.

Creativity is encouraged, but clarity and feasibility are essential.
