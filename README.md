# Komently — AI-Ready Universal Comment System

Komently is an AI-ready universal comment system designed for modern website builders. It provides a complete infrastructure for comment sections including a performance-first threaded UI, interactive engagement tools, and a unified management dashboard.

This project is a draft working version built with Next.js, demonstrating the core concepts and preparing the foundation for future AI-agent integration as part of Homework 2: Website Development & AI Agent Planning.

## 🚀 Key Features

- **Universal Integration:** Single-snippet integration for any web stack.
- **Modern Threading:** Performance-optimized reply system for high-engagement discussions.
- **Interactive Engagement:** Built-in upvotes, downvotes, and real-time reactivity.
- **Developer Dashboard:** Centralized command center to manage multiple properties and moderation settings.
- **AI-Ready Architecture:** Designed specifically to integrate autonomous moderation agents.

## 📦 Setup and Run Instructions

### Prerequisites
- Node.js (v18 or later)
- npm, yarn, pnpm, or bun

### Local Development
1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Komently
   ```

2. Navigate to the website directory:
   ```bash
   cd website/next-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.
